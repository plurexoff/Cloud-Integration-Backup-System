## Основные функции и классы

### GitHubCloudManager

```python
class GitHubCloudManager:
    def __init__(github_token: str)
    def initialize_backup_repo(repo_name: str) -> bool
    def upload_file(local_path: str, cloud_path: str) -> (bool, str)
    def download_file(cloud_path: str, local_path: str) -> (bool, str)
    def backup_directory(local_dir: str, cloud_dir: str) -> Dict
    def restore_backup(cloud_dir: str, local_restore_path: str) -> Dict
    def list_backups(base_dir: str) -> List[Dict]
    def list_files(cloud_path: str) -> List[Dict]
    def delete_file(cloud_path: str) -> (bool, str)
    def get_repo_info() -> Dict
```
